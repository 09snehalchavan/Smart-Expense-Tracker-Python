# 💰 Smart Expense Tracker

## 📌 Project Overview

Smart Expense Tracker is a Console-Based Python application developed to help users efficiently manage their daily income and expenses.

The application stores data securely using SQLite and provides various financial management features such as budget tracking, monthly reports, category-wise analysis, and report exporting.

---

## 🚀 Features

* User Registration
* User Login Authentication
* Add Income
* Add Expense
* View All Transactions
* Search Transactions
* Monthly Report
* Category-wise Expense Analysis
* Budget Management
* Export Report
* View User Profile
* Logout
